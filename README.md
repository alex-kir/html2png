# html2png

```
Tool renders HTML into PNG

Options:
--help, -h              Boolean.
--input, -i             String. Input HTML url
--output, -o            String. Output PNG filename
--width, -w             Int32. Browser window width
--height, -h            Int32. Browser window height
--dpi                   Int32. Browser window DPI
--cache-path            String. Cache path
--delay                 Double. Delay, seconds
```
